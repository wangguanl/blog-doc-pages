产品介绍 #

乐高是高效、高性能的拖拽式无代码开发平台，通过行业化模板、拖放式组件和可视化配置快速构建专题，将繁琐的底层架构和基础设施抽象化为图形界面，免去了代码编写工作，能够完全专注于业务场景。将搭建的应用与物料打通，提供高度开放的操作环境。可实现业务疾速交付，降低开发成本，来代替传统的手工编码计算机程序。为企业提供低门槛、疾速构建多端的服务。

解决了什么问题 #

去中心化：

  • 随着企业的高速发展，大量的业务开发需求接踵而来；像人手紧缺、重复开发、沟通效率低下等问题，暴露得愈发明显；传统的开发模式周期长而成本高。
  • 大幅缩短制作周期、降低开发成本，提高企业生产力。
  • 让IT团队由典型的 **成本导向型组织 **转型 价值输出型组织
  • 没有依赖关系，可共享物料

对比其他平台 #

其他平台是乐高的灵感来源，然而其他平台做的更多是low-code（低代码），而**no-code（无代码）**是乐高的存在的意义。汲取了其他平台的优点，很多亮点已经得到了改进，更符合企业的使用场景。

其他平台 #

  1. 不符合企业现有业务流程
  2. 不仅仅是无代码，还主要具备如下四个特性：物料接入、编排能力、渲染能力、出码能力。技术介入：美恰对话通讯工具、gio埋点等易于扩展业务
  3. 使用人群要求高，要求用户有专业知识，使用场景复杂，学习成本高

乐高 #

  1. 低门槛：
    • 无论你懂不懂代码，无论你是技术岗还是其他岗位，简单的拖拽应用即可搭建专题；
    • 内置了非常多的模板和自定义组件可直接使用，在模板的基础上进行修改后就可以上线。
  2. 易用性：由于与其他平台有着众多的相似处，我们在改进时在用户体验这一步花费了很多的功夫，让用户使用起来更加便捷
  3. 灵活性：乐高是更加灵活开放的解决方案。简单易用但这并不意味着乐高的功能不强大，它属于渐进式，由浅入深。
    • 对于初级使用者来说不是必须的，但想要做出更丰富的场景，对中高级使用者很有必要。
      1. 通过样式面板调整组件的样式，让组件使用场景更加丰富。
      2. 通过自定义组件的方式搭建可重用组件，从两个应用中提取出一些公共的片段组成自定义组件，以达到复用的目的。
    • 当你掌握更多内容时，你做的会更好，做出跟多想要的内容，更有趣的内容。这意味学习成本也会增加。
  4. 拓展性：为技术人员提供可向上扩展（Schema）和向下扩展（HTML）。
  5. 多 端：目前支持 **PC ** 和 **移动 ** 两端

功能 #

由技术经理带队发起项目实施，一款产品设计到开发功能再到项目落地测试，技术全权驱动整体流程，共计30个工作日 #

组件列表 #

  • 基础组件：文本、超链接、咨询热线、在线客服、顶部导航、底部导航、

  • 媒体组件：图片、视频、轮播图、

  • 表单组件：输入框、按钮、

  • 自定义组件：用户可根据需求抽离出组件达到复用的目的

画布 #

  • 组件

    • 鼠标选中组件：使用 Ctrl + C 组合键复制， 使用 Ctrl + V 组合键粘贴
    • 可进行 拖拽移动、拖拽缩放

  • 画布

    • 鼠标点击画布按 Shift 选中：使用 Ctrl + + 组合键放大，使用 Ctrl + - 组合键缩小， 使用 Ctrl + 0 组合键恢复100%原比例
    • 可进行 拖拽移动
